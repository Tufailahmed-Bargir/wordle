## How to run?

### Manual setup

1. Fork this repo and make a clone of it and go to the **cd lego** directory
   ```
   cd wordle
   ```
2. make **pnpm install**
3. ```
   pnpm install
   ```
 
5. Run/ Start localhost by (`pnpm run dev`)

```
pnpm run dev
```

7. then visit to **`localhost:3000`** in your browser.
